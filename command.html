📋 Onboarding Playbook

Quick reference — how to onboard every type of person.

Type 1: New Person — Paid

"Sarah wants in, she's paying $1K Launch Special"
1. Go to My Codes → fill out request (name, package, VPN, payment status, email)
2. Submit to SBA HQ → Keith confirms money → approves → codes push to you
3. Click ⚡ Onboard on her card → form pre-fills
4. Pick who referred her → click Onboard → copy welcome email → send
⏱️ 2 minutes

Type 2: Free Affiliate (no payment)

"Marcus wants to recruit but isn't paying yet"
1. Go straight to Onboard Member tab — NO code needed
2. Member Type: Affiliate Only
3. Fill in name + email → pick referrer → Onboard
4. Marcus can recruit, commissions HELD until he pays. No Genius access. 40 days to convert.
⏱️ 1 minute

Type 3: Legacy (already paid before)

"Tanya was with us before, already paid Keith $2K"
1. Submit request → note "Legacy — already paid"
2. Keith approves → onboard with Member Type: Legacy
3. Set Affiliate Type: Grandfathered (55% rates)
4. Full access, cycle starts March 25, must renew at full price May 4
⏱️ 2 minutes

Type 4: Phantom (not here yet, has people)

"Mike isn't signed up but George, Robert, Taylor are his people"
1. Type "Mike Johnson" in pending upline field → click + Add as Pending
2. Mike appears in dropdown as ⏳ pending
3. Onboard George, Robert, Taylor → select Mike as their referrer
4. When Mike shows up → Keith converts him from phantom to active
5. No commissions flow through Mike until he's activated
⏱️ 1 minute per person

Type 5: Jumping Ahead (upline not here)

"Sarah wants in but her upline Mike isn't in the system"
1. Create Mike as phantom first (+ Add as Pending)
2. Onboard Sarah → select Mike (pending) as referrer
3. Tree is correct from day one — no moves needed later
⏱️ 1 minute

Type 6: Partial Info (no email yet)

"John wants in but I only have his name"
1. DON'T onboard yet — minimum needed: name + email
2. Submit request with ❌ Not Paid, note "getting email tonight"
3. Keith holds the request until info is complete
4. When you have email + payment → submit new request or tell Keith
⏱️ Wait for info

⚡ Quick Reference — 7 Questions

1. What's their name?
2. What's their email?
3. Are they paying or free affiliate?
4. If paying — what package? Got VPN?
5. Who brought them in? (referrer)
6. Is their upline in the system? → Yes: select. No: create phantom.
7. Did the money come in? → Yes: ✅ Paid. No: ❌ Hold.
